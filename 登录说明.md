# 登录说明

## 启动项目

在项目根目录运行：

```bash
npm run dev
```

启动后打开：

```text
http://localhost:3000
```

## 进入登录页

可以通过两种方式进入：

1. 打开首页后点击“进入演示系统”
2. 直接访问：

```text
http://localhost:3000/login
```

## 当前登录方式

当前版本是演示登录，不需要账号密码。

在登录页直接选择角色即可：

- 学生端：进入学生工作台和引导式问答
- 家长端：进入作业上传、批改结果和辅导建议
- 教师端：进入学情看板、学生列表和教学建议

## 切换身份

如果要切换角色：

1. 点击页面右上区域的“退出演示登录”
2. 回到登录页重新选择身份

## 补充说明

- 系统使用浏览器 cookie 记录当前演示身份
- 如果页面访问异常，先确认开发服务仍在运行
- 更完整的项目说明见 `README.md`
